## Run it yourself

Declare an environment variable containing the bing maps key:

In Windows:

    setx BING_MAPS_KEY ...

Or using the UI.

In Linux / Mac 

    export BING_MAPS_KEY=...
    
(you can put that in your .profile / .zshrc)

```bash
# install dependencies
$ npm ci

# serve with hot reload at localhost:3000
$ npm run dev
```
